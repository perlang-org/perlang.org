Class Binding

Holds information about a binding.

Different types of bindings (=subclasses) provide slightly different mechanisms to retrieve information about the variable, function or class being bound to.

ReferringExpr

Gets an expression referring to the declaring statement's type reference. Note that multiple expressions can refer to a single declaration statement, as illustrated by the following program:

var a = 123;
var b = a; // b refers to a
var c = a; // c also refers to a.
