Class NameResolver

The NameResolver is responsible for resolving names of local and global variable/function names.

Type Name Description

A dictionary of global classes, with the class name as key.

A callback used to add a global, top-level class.

A callback which will be called in case of name resolution errors. Note that multiple resolution errors will cause the provided callback to be called multiple times.
